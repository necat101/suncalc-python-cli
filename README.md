# SunCalc Python CLI

A Python CLI wrapper around the JavaScript suncalc library for calculating sun and moon positions, rise/set times, and phases.

## Quick Start

```bash
git clone https://github.com/necat101/suncalc-python-cli.git
cd suncalc-python-cli
npm install
pip install geopy
python3 suncalc_cli.py --city "Stephens City, VA"
```

See README.md for full documentation.
